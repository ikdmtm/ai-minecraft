6

MANUAL モード: 配信テスト

systemctl start orchestrator → ダッシュボード (SSHトンネル :8080) から「配信開始」。
死亡したら配信終了 → 停止（次は始めない）。

手動開始死亡で停止
7

AUTO モード: 24時間自動配信

OPERATION_MODE=AUTO に切替 → orchestrator 再起動。
死亡 → 教訓生成 → 60秒クールダウン → 新配信自動開始。
1日最大20回。上限で翌日まで休止。

全自動24/7
